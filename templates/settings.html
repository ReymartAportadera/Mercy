What is being monitored?
The system monitors ALL drives and user folders including Desktop, Downloads, Documents, and any external drives. Files are scanned automatically when created, modified, or moved.
Real-Time Protection Features:

✅ Full System Monitoring - All drives and user folders
✅ Instant Detection - Files scanned immediately when created
✅ Smart Filtering - Auto-skips system files and caches
✅ Auto-Quarantine - Critical threats moved to quarantine
✅ Desktop Notifications - Windows toast alerts for threats
